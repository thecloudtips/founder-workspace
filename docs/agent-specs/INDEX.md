# Agent Deployment Specifications

Detailed agent architecture documents for plugins with complex multi-agent patterns. Each plugin folder contains PRD, architecture, tech-spec, and deployment docs.

## When to Add

Create agent specs for plugins that use multi-agent patterns (Pipeline, Parallel Gathering, Competing Hypotheses) and need detailed coordination documentation beyond what SKILL.md provides.
